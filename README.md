# MOVIE DB

A Personal Movie DB using themoviedb.org API service to search for and provide information about movies

---
Application Package
--------
1. remember to add MONGODB_URI environment variable
2. mvn clean install
3. java -jar target/moviedb-0.0.1-SNAPSHOT.jar
